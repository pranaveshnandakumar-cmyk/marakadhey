# Marakadhey - Don't Lose Opportunities

Marakadhey is a simple Chrome extension that helps you save important webpages and receive reminders before deadlines.

Whether it's an internship application, coding assignment, scholarship, webinar, hackathon, certification, registration, or any important opportunity, Marakadhey ensures you don't forget it.

## Features

✅ Save the current webpage with one click

✅ Automatically capture page title and URL

✅ Set reminder date and time

✅ Organize reminders by priority

✅ Smart Inbox for tracking pending and completed reminders

✅ Desktop notifications with snooze support

✅ Dashboard with reminder statistics

✅ Optional Google Calendar integration for mobile notifications

✅ Mark reminders as completed after opening

✅ Hide completed reminders to keep your inbox clean

## Privacy

Marakadhey stores reminder data locally on your device using Chrome Storage.

No personal information is collected, transmitted, sold, or shared with third parties.
